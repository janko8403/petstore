# Petstore Laravel 11 Application

This is a simple Laravel application for managing a pet store.

## Installation Instructions

### Prerequisites

- PHP >= 8.2
- Composer
- MySQL
- Node.js & npm

### Steps

**Clone the repository:**
    ```bash
    git clone https://github.com/janko8403/petstore.git
    cd petstore
    ```


### Additional Commands

- **Run tests:**
    ```bash
    php artisan test
    ```
